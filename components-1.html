Component Hub

What is a component?

A component is a piece of web design/technology that you can use in your page to either convey additional information or customize it to your liking. They can range from the SD Class to a simple infobox. If you’re looking to spice up your page with some fancy tech or help direct your readers somewhere, you’re in the right place!

How do I use components?

A component page will always specifically tell you how to use it in your page, but generally, it’ll look like this:

[[include :backrooms-wiki:component:namehere]]

A component may also have other parameters you need to specify in order to use it, but that will always be included in the page.

How do I post my own component?

Check out our Greenlight Policy! In summary, all components must be approved by one of our Techlighters before you can post it. There is also coding help available in the Discord server and the site forums.

Once you get the go-ahead, you’ll want to make sure the url has component: at the beginning and tag it with the relevant tags so that they show up here automatically. The tags are as follows:

Tag Criterion
component This tag is for all components.
common-component This tag is for components that are heavily used throughout the wiki and may even be required to use in a page.
class-component This tag is for components that are for any kind of classification.
aesthetic-component This tag is for components that are for customizing a page and adding aesthetic value.
goi-component This tag is for components that are tied to a GOI.


If you don’t want your component to show up on this hub, tag it with dont-show.
