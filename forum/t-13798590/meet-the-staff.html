CutTheBirch 09 May 2021 01:56

To be a staff, you need to be in the discord server if you aren't already. The link to join is on the front page (even if you're active on the forums, being on discord is necessary for staff communication). You should also probably be active in the discord server for at least a couple weeks before applying so that the staff can get to know you, and see if you would be a good pick.

For greenlighter, Discord isn't necessary. That one is basically based more on how good you are at critiquing and writing. So, I would basically have to see some examples of some crit or writing from you. DM me if you're interested.

CutTheBirch 15 Feb 2022 22:44

It is possible to become a greenlighter without discord, but it's highly recommended that you join discord if you want to become a greenlighter because you need to be on discord to access the Greenlighter Mentor Program, which is a server where greenlighters help non-greenlighters improve on their crit until they're ready to become greenlighters themselves.

It is still possible to become a greenlighter if you can't go on discord for whatever reason.

Either way, DM me, and I can help you.
