Tag Suggestion Thread
VivamusLudio 31 Aug 2023 00:45

Hello, and welcome to Tag Suggestion Thread, a Thread for you to Suggest Tags.

When proposing a tag, please make sure to check if it fits the requirements shown in the Tag Guide first, and to use the following template:

**Name:** Tag name.
**Relevant pages:** A list of all pages that would qualify for this tag.
* [[[PAGE NAME 1]]] by [[*user author1]]

X pages by Y authors.

Staff will review proposals and edit the original post to signal if the suggested tag's implementation has been accepted or denied.
