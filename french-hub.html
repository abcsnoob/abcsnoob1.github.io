This page is a collection of English translations of French originals which are not written originally for this site but for the French site. If you are a French-speaking user, it's worth your time to visit the French site to read there and participate!

Read the language FAQ on the Translation Hub if you have any questions about this page.

The French language custodians are:

  • CamaradeAlbabar

Some sites may have a different minimum age requirement for example — don't assume you're eligible to join it just because you're a member of this site!

Backrooms Wiki FR — Les Backrooms

http://fr-backrooms-wiki.wikidot.com/

This is the Language Wiki for French versions of English Backrooms content, but is also the main publisher of French originals! This site has a major focus on developing its own unique lore including original PoIs and groups. Come and read some French Backrooms pages!

Established: 2021
Originals / Translations: This site hosts both originals and translations.
Contactable Staff: Praetor3005, Mel-Raspberry
