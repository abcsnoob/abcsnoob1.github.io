Description and Characteristics

The Back OS™ logo. It's identical to that of Windows 95, except that only the yellow color is used. Maybe it's the work of one of us? Or, maybe some kind of entity? Or could it be the OS itself? Who knows…

Object 94, commonly referred to as Back OS™, is an operating system famous for being the only functional OS on the BackNET1, as other operating systems such as Windows or Linux cannot work with the Wi-Fi2.

If you're wondering, Back OS™ is obviously a made-up name. In reality, no one knows its real one, as the OS is only represented by its logo. Given that no apparent “official” name could be found for Object 94, The Leaders3 published a survey on the BackNET in 1990, where "Back OS™" was featured among the multiple choices. This generic yet strangely memorable name gained massive popularity and ended up winning the poll with 78% of the vote, and from that day on, everyone refers to Object 94 as “Back OS™”.

Back OS™ is installed by default on all Backrooms-native computers and connected devices, with no possibility of replacing it or modifying its source code. It is however possible to install it on "Frontrooms devices" by various means, but this action is irreversible as it will permanently replace your current operating system.

Back OS Mobile™

Back OS™ can also run on mobile and smaller devices such as tablets, with what looks like an adapted version. The features it offers and the way it works remain broadly the same; only the interface is somewhat different to suit a smaller screen.

A Mobile Phone Booting Back OS™.

How to Install Back OS™

Step 1: Get the source files

If you want to install Back OS™ on your "Frontrooms device", you'll need a USB flash drive or a hard disk drive containing Back OS™4. These objects can be found in office spaces such as Level 4 and office buildings in Level 11, or inside tech stores and tech aisles of supermarkets. You can also find these objects in technological levels like Level 102, Level 502, Level 0.1 or Level 4005. Otherwise, these items can be traded with the B.N.T.G. or The Backrooms Robotics, but the prices are far too high for what it is.

Object-94-A: A Flash Drive containing Back OS™.

Object-94-B: A HDD containing Back OS™.

Step 2: Proceed with the installation

The OS recognizes device types and automatically selects the appropriate version in case of an installation:

  • For computers, plug a USB Flash Drive or insert an HDD containing Back OS™. Next, press F12 at boot time and select “Boot by USB” or "Boot by Disk" - Back OS™ will then install itself.
  • For smartphones, you'll have to find a variant of Object 94-A with a USB-A or USB-C connector. Plug it and an installation agent will open, asking if you want to install Back OS™ on your phone. Confirm and the OS will install itself.

Other Devices

We recently discovered that Back OS™ affects many more device types than we originally thought.

Indeed, one of our experiments was to determine whether the operating system would work on a connected refrigerator, and to our surprise, it actually did! But it doesn't add anything special, I'd even say it's a waste of time to install Back OS™ on it… I mean, no offense, but who writes their emails on a refrigerator anyways?6

Back OS™ Account

Once you boot up a Back OS™ powered device, you will be prompted to log in with a USERNAME and a PASSWORD. If this is the first time you're logging in, you'll have to create an account7. Once the account has been created, you can use it to connect to another Back OS™ powered device, restoring your session and everything you did on the previous device. This allows you to access your files anywhere in the Backrooms, and to not worry about your device being destroyed or broken. Upon closer inspection, it has been revealed that the OS features a cloud service named The BackRoom™ that automatically saves what you're doing on your account.

Features and Default Applications

Back OS™ appears to be a mix of Windows 95 and Windows XP (the two most popular operating systems of their respective eras) in their features and their rather simplistic interface.

As a result, Back OS™ offers most of the applications and features that were available on those operating systems.8

Just like “Back OS™”, the default applications only have their logo to identify them, meaning that the names of the following applications have been chosen using the same naming process mentioned earlier. This is mainly intended to establish a universal slang and simplify communication. On a side note, the operating system allows you to rename applications on the desktop to whatever you like, meaning the names below are simply the most common ones.

BackNET Explorer™ can mainly be used to access and get information from the General Public Database, in case you are lost and want to know about a specific level, object, or entity.

Websites are hosted on remote servers which locations remain a mystery to this day. HTTPS is used as a protocol to encrypts data passing between websites, and therefore, user data passing through the BackNET. Since users can publish their own websites, there are of course "official" websites published by the search engine itself, such as the infamous "Wikihow: How to exit the Backrooms in 7 steps." The official websites are identified by the Back OS™ logo next to the link, which cannot be replicated if the website is posted by a user.

Network and Cybersecurity

Offline Mode

Back OS™ can still work without Wi-Fi in case you don't have access to it. However, most of the features and services including The BackRoom™, BackNET Explorer™, etc. won't work. You can also disconnect and connect to the Wi-Fi at will, which is actually useful in some situations…

Main network and subnet

As mentioned in the BackCode Studio™ section, the BackNET seems to divided into two distinct networks. The main one that uses unknown protocols (used by every default features and applications) and the subnet (only used by user-made websites on the BackNET) that uses IPv4.

But since the year 2004, it has been revealed that the second network is vulnerable to cyber attacks. However, these attacks are limited to the IPv4 network, and do not extend to the main network.
