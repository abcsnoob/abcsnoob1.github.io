Wikiproject No-Ifs

This project is primarily run by Abdallah Amr, contact me if you have any questions.
All you need to to do is add ">[!-{$-}" before the beginning [[iftags]] block, and add "[!-- --]" after the end of it. If the page happens to have one or more include clauses in it, add "-={$-}" at the end of the first line of each include clause, so for example:

[[include :backrooms-wiki:component:level-class -={$-}
|class=2
]]

Oh also, the tag to tag these pages with is _no-ifs
