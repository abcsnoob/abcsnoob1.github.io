infoHow to add a theme?

When you make your page, add one of the tags that correspond to the theme category. Then it'll auto_awesomeautomagicallyauto_awesome be added to the theme page. Add as many of them that apply, except for that you shouldn't add aesthetic-theme if any tags other than goi-theme and canon-theme are applied (since aesthetic is a catch-all).

+ Tags
- Tags
  • site-theme - a close version of the main site theme
  • goi-theme - tied to a goi
  • canon-theme - tied to a canon
  • aesthetic-theme - not tied to a canon or goi, just looks nice
  • digital-theme - looks like a computer screen
  • monochrome-theme - has only one primary colour throughout the theme
  • paper-theme - looks like a physical page instead of a web page

aesthetic-theme is the catch-all for themes which don't fall in any other category!

Then, so that the click to preview button works, you need to make a fragment page for the theme. If your theme's link is theme:kalag, then your fragment page should be fragment:kalag. What goes into your fragment page is as simple as the theme's include, and anything else you need to add to make the preview work properly:

[[include :backrooms-wiki:theme:kalag]]

[[module css]]
/* Additional CSS goes here. */
[[/module]]
